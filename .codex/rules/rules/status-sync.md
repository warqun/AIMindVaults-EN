# Codex Root Status Sync

> Root status is for multi-vault routing context only.

## Rules

- Record root-level decisions in `.codex/AGENT_STATUS.md`
- Record vault execution work in the target vault's `.codex/AGENT_STATUS.md`
- When checking active work, prefer the target vault `_STATUS.md` plus both agent status files in that vault
- Do not use root status as a substitute for per-vault execution history
