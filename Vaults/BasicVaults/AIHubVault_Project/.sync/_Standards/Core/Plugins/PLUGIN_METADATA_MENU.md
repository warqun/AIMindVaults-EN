# Metadata Menu

## 기능

- frontmatter/필드 값을 UI로 수정
- 다수 문서 메타데이터 일관화

## 주 사용 작업

- `status`, `priority`, `epic`, `system` 입력/수정
- 문서 상태 일괄 정리

## 기본 절차

1. 필드 스키마(키 이름) 확정
2. 문서에서 필드 UI로 값 입력
3. Dataview 결과로 반영 여부 확인
