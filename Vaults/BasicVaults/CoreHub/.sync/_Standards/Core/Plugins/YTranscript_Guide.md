# Youtube Transcript (YTranscript) 플러그인 사용 가이드

**목적**: 유튜브 영상(강의, 튜토리얼 등)의 스크립트와 타임스탬프를 옵시디언 노트로 바로 추출하여 원활한 영상 학습 및 메모를 돕습니다. 파이썬 스크립트 작성이나 봇 우회를 신경 쓸 필요 없이 옵시디언 내에서 모든 작업이 해결됩니다.

---

## 🛠 주요 기능 및 장점

- **모바일/데스크톱 완벽 지원:** PC뿐만 아니라 모바일에서도 부드럽게 작동합니다.
- **클릭 가능한 타임스탬프:** 추출된 스크립트 옆에 `00:15` 형태로 타임스탬프가 삽입되며, 이것을 클릭하면 영상의 해당 시간대로 바로 점프합니다.
- **사이드 패널(Classic) 지원:** 우측 사이드 탭에 스크립트를 띄워놓고 보면서 메인 창에 자기만의 요약 노트를 작성하기 좋습니다.

---

## 🚀 사용법 요약

### 1. 인라인(인서트) 방식 (추천: 빠른 노트 작성용)
노트 본문에 곧바로 전체 스크립트를 삽입하는 방법입니다.

1. 삽입할 곳에 커서를 둡니다.
2. 유튜브 영상의 URL 주소를 복사합니다.
3. 명령어 팔레트(`Ctrl` + `P`)를 열고 아래 명령어를 실행합니다:
   > **`Youtube Transcript: Insert Youtube transcript`**
4. 팝업창에서 URL을 확인하고 엔터를 치면, 번역이 완료된 스크립트와 타임스탬프가 내 노트에 바로 꽂힙니다.

**(참고)** 다음과 같은 순서대로 URL을 캡처하여 작동합니다:
- 현재 노트에 지정(드래그)된 텍스트
- 시스템 클립보드(복사해둔 주소)
- 수동 입력 프롬프트(둘 다 없을 경우)

### 2. 사이드 패널 방식 (추천: PC에서 집중 학습용)
노트 작성과 원문 참고를 분리하고 싶을 때 사용합니다.

1. 에디터에서 유튜브 비디오 링크를 선택합니다.
2. 팝업창(또는 우클릭 메뉴 등)에서 다음 명령어를 실행합니다:
   > **`Youtube Transcript: Get Youtube transcript from selected url`**
3. 사이드 패널 창이 열리며 타임스탬프가 달린 텍스트가 표시됩니다.
4. 패널 안에서 특정 줄을 노트 쪽으로 드래그 앤 드롭해서 일부 내용만 삽입할 수 있습니다.
5. 설정(Settings) 메뉴에서 타임스탬프가 표시되는 주기(예: 매 32줄마다 표시)를 바꿀 수 있습니다.

---

## 💡 활용 팁
- 스크립트를 그대로 복사해 온 뒤에, GPT나 Claude 등 강력한 언어 모델 프롬프트(또는 관련 플러그인)를 통해 **'핵심 내용 3줄 요약'** 등의 지시를 내려 빠르게 내용을 소화하는 용도로 연결하면 최상의 효율을 얻을 수 있습니다.
