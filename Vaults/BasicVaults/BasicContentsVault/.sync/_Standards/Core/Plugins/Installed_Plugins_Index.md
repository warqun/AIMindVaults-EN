## Plugins Requiring BRAT Reinstall (When Mirror Cloning)

When mirror-cloning a vault, the following plugins are not in the official community plugin list or use specific branches/beta versions, so they must be **reinstalled via BRAT using the original repository links**.

| Plugin Name | Repository | Reinstall Git Link |
|:------------|:-----------|:--------------------------------------------------|
| **Time Machine** | `dsebastien/obsidian-time-machine` | https://github.com/dsebastien/obsidian-time-machine |
| **YTranscript**  | `lstrzepek/obsidian-yt-transcript` | https://github.com/lstrzepek/obsidian-yt-transcript |
| **Claudian**     | `YishenTu/claudian`                | https://github.com/YishenTu/claudian                |

> [!tip] How to install beta plugins via BRAT
> 1. Install and activate the **BRAT** plugin in Obsidian.
> 2. Go to BRAT settings and click the **Add Beta plugin** button.
> 3. Copy the `Reinstall Git Link` (or repository name) from the table above and paste it into the input field.
> 4. Click Add Plugin to complete the installation, then activate the plugin in the Community plugins tab.
