## ⚠️ BRAT 재설치 필요 플러그인 (미러 복제 시)

미러 복제(Vault 복제) 시, 아래의 플러그인들은 정식 커뮤니티 플러그인 목록이 아니거나 특정 브랜치/베타 버전을 사용 중이므로 BRAT을 통해 **본래 레포지토리 링크로 재설치**해야 합니다.

| 플러그인명            | 리포지토리명                             | 재설치용 Git 링크                                         |
| :--------------- | :--------------------------------- | :-------------------------------------------------- |
| **Time Machine** | `dsebastien/obsidian-time-machine` | https://github.com/dsebastien/obsidian-time-machine |
| **YTranscript**  | `lstrzepek/obsidian-yt-transcript` | https://github.com/lstrzepek/obsidian-yt-transcript |
| **Claudian**     | `YishenTu/claudian`                | https://github.com/YishenTu/claudian                |

> [!tip] BRAT을 통한 베타 플러그인 설치 방법
> 1. Obsidian에서 **BRAT** 플러그인을 설치 및 활성화합니다.
> 2. BRAT 옵션 설정으로 이동하여 **Add Beta plugin** 버튼을 클릭합니다.
> 3. 위 표의 `재설치용 Git 링크` (또는 리포지토리명)를 복사하여 입력란에 붙여넣습니다.
> 4. Add Plugin 버튼을 눌러 설치를 완료한 후, 커뮤니티 플러그인 탭에서 해당 플러그인을 활성화합니다.
