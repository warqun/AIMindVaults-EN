# Tasks

## Features

- Checkbox-based task management / search
- Execution lists by due date / status

## Main uses

- Track priority work for today / this week
- Report open tasks

## Basic syntax

```tasks
not done
path includes docs
sort by due
```

## Recommended rules

- When creating a task, at minimum record `assignee / due / linked note`.
