# Templater

## Features

- Automatic note template generation
- Date/project/status default value injection

## Primary Use Cases

- Issue/meeting notes/spec document creation
- Frontmatter standardization

## Basic Procedure

1. Select template from `_Standards/NoteTemplates`
2. Create note from template
3. Verify required fields (`type`, `status`, `updated`)
