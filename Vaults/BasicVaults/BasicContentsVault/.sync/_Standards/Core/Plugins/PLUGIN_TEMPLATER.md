# Templater

## 기능

- 노트 템플릿 자동 생성
- 날짜/프로젝트/상태 기본값 주입

## 주 사용 작업

- 이슈/회의록/스펙 문서 생성
- frontmatter 표준화

## 기본 절차

1. `_Standards/NoteTemplates` 템플릿 선택
2. 템플릿으로 노트 생성
3. 필수 필드(`type`, `status`, `updated`) 확인
