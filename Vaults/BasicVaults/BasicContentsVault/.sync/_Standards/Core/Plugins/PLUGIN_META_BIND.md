# Meta Bind

## Features

- Bind in-note forms to metadata
- Human input auto-updates frontmatter

## Main uses

- Build checklists / decision forms / request forms
- Unify collaborative input interfaces

## Basic procedure

1. Define standard field keys
2. Write the input form block
3. After saving, verify frontmatter values are in sync
