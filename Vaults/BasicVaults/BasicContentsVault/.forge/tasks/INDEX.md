# _forge/tasks — External Agent Task Drafts

Prepare tasks to send to Grok / Gemini / Codex for AgentForge development here.

## File Name Convention

| File Name | Purpose |
|-----------|---------|
| `TASK_GROK_[topic].md` | Task instructions for Grok |
| `TASK_GEMINI_[topic].md` | Task instructions for Gemini |
| `TASK_CODEX_[topic].md` | Task instructions for Codex |

## Rules

- Grok prompts from `/grok-route` results may be saved here
- Completed tasks get a `_SENT` suffix added to the file name
