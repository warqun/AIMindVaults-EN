# BasicDiaryVault

> Diary Preset Hub (`AIHubVault_Diary`) 용 위성 클론 소스 템플릿. 일기 전용 폴더 구조 (`Contents/Daily`, `Weekly`, `Monthly`, `Yearly`, `Reviews`, `_AI_Drafts`, `_archive`), Templater 템플릿 5종, Dashboard Dataview 뷰를 포함한다.

## 사용법

이 볼트를 직접 편집하지 않는다. `C:/AIMindVaults/Vaults/BasicVaults/MakeCloneVault.bat` 실행 → Diary Hub 선택 → 새 위성 볼트가 생성된다.

- 스펙: `Vaults/Projects_Infra/Project_AIMindVaults/Contents/Project/plan/architecture/20260420_Diary_Hub_명세.md`
- 초기 설정·프라이버시 원칙은 `CLAUDE.md` 참조.
