# QuickAdd

## Features

- Shortcut execution for capture/template/command chains
- Automate repetitive routines (daily/issue creation)

## Primary Use Cases

- Quick TODO capture
- Ultra-fast creation of same-format notes

## Basic Procedure

1. Define action (Template/Capture/Macro)
2. Bind to hotkey or Command Palette
3. Verify auto-populated links/tags in created results
