# Tasks

## Features

- Checkbox-based task management/search
- Due date/status-based action list generation

## Primary Use Cases

- Today's/this week's priority task tracking
- Incomplete task reports

## Basic Syntax

```tasks
not done
path includes docs
sort by due
```

## Recommended Rules

- When creating tasks, always include at minimum: assignee, due date, and linked note.
