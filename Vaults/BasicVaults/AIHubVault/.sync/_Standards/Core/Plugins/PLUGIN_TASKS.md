# Tasks

## 기능

- 체크박스 기반 작업 관리/검색
- 마감일/상태 기반 실행 목록 생성

## 주 사용 작업

- 금일/이번 주 우선 작업 추적
- 미완료 작업 리포트

## 기본 문법

```tasks
not done
path includes docs
sort by due
```

## 권장 규칙

- 태스크 생성 시 최소 `담당/마감/연결노트`를 같이 기록한다.
