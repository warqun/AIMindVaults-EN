## ⚠️ Plugins that require BRAT re-install (after mirroring)

When you mirror (clone) the vault, the plugins below are not in the official community list or use a specific branch / beta build, so they must be **re-installed via BRAT using the original repository link**.

| Plugin | Repository | Git link for re-install |
| :----- | :--------- | :---------------------- |
| **Time Machine** | `dsebastien/obsidian-time-machine` | https://github.com/dsebastien/obsidian-time-machine |
| **YTranscript** | `lstrzepek/obsidian-yt-transcript` | https://github.com/lstrzepek/obsidian-yt-transcript |
| **Claudian** | `YishenTu/claudian` | https://github.com/YishenTu/claudian |

> [!tip] Installing a beta plugin with BRAT
> 1. Install and enable the **BRAT** plugin in Obsidian.
> 2. Open BRAT settings and click **Add Beta plugin**.
> 3. Copy the `Git link for re-install` (or repo name) from the table above and paste it into the input.
> 4. Click Add Plugin to finish, then enable it in the Community plugins tab.
