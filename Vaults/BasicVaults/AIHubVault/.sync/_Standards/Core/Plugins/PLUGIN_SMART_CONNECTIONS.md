# Smart Connections

## 기능

- 의미 기반 유사 노트 탐색
- 관련 문서 추천으로 누락 참조 보완

## 주 사용 작업

- 새 문서 작성 전 선행 자료 수집
- 중복 문서 탐지

## 기본 절차

1. 핵심 키워드가 있는 노트 열기
2. 추천 노트 목록 확인
3. 참조할 노트를 본문 링크로 명시

## 주의

- 추천 결과는 보조 신호이므로 최종 판단은 수동 검토로 확정한다.
