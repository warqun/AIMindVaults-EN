# _forge/tasks — External Agent Task Directive Drafts

Prepare tasks to send to Grok / Gemini / Codex for AgentForge development here.

## File Naming Convention

| Filename | Purpose |
|----------|---------|
| `TASK_GROK_[topic].md` | Task directive for Grok |
| `TASK_GEMINI_[topic].md` | Task directive for Gemini |
| `TASK_CODEX_[topic].md` | Task directive for Codex |

## Rules

- Grok prompts from `/grok-route` results may be saved here
- Add `_SENT` suffix to filename once the task has been dispatched
